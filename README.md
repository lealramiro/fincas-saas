# Fincas SaaS - B2B Property Management Platform

SaaS platform for property administrators (comunidades de propietarios) 
to automate incident management and daily operations.

## Live Demo
https://fincas-saas-nine.vercel.app

## Stack
Next.js 15 · TypeScript · Prisma · Tailwind CSS · Supabase (PostgreSQL) · Vercel

## Features
- Incident management and tracking
- Role-based access control
- Responsive dashboard for property administrators
- Automated workflows for common operations

## Context
Final degree project (TFG) - Double Bachelor in Computer Engineering 
and Business Administration, Universidad Carlos III de Madrid (UC3M).
Business plan and technical specification developed alongside the prototype.
